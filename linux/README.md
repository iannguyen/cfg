## Order of Operations

##### Scripts
- 'post_install.sh'
- 'core_install.sh'
- 'dev_install.sh'

##### Extras

- add `noatime` for SSDs, in `/etc/fstab`, in front of `errors`
  - i.e `noatime,errors=remote-ro`
- Grub Customizer
- Unity Tweak Tool
- Gnome Tweak Tool
- Touchpad-Indicator
- Dash-to-dock extension
- imwheel, copy the .imwheelrc to the root directory
  - set to autostart
  - `imwheel -b "4 5"`
- RedShift
